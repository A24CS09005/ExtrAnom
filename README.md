## ExtrAnom Dataset
**ExtrAnom** is a multi-modal dataset containing 1,001 videos with textual descriptions, including 500 normal and 501 anomalous videos. The anomalous category focuses on incidents involving women, such as chain snatching, stalking, harassment, kidnapping, and assassination. Each folder in the dataset contains the corresponding video files along with an `annotation.txt` file. To utilize the annotations, the `annotation.txt` file can be converted into JSON format for direct use. The dataset will be made available upon request for research purposes.
